Plugin that displays Your Posts as a Live News Bar.

== Description ==

It is very light weight and doesn't too many javascript codes, just give user option to share or follow!

+ This plugin will show <a href="http://www.liket.ly/likebutton/">LiketLy Bar</a> on your website
+ Auto Post a <a href="http://www.liketly.com">LiketLy</a> Update (Post title + Link) when you publish/update your Wordpress blog.
+ Demo: <a href="http://www.expertcustomize.com/demo/arowana/">Arowana Blog</a>

== Installation ==

1. Upload plugin zip file to the `/wp-content/plugins/` directory
2. Register a <a href="http://www.liket.ly">www.Liket.ly</a> profile.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Appearance > Liketly Bar Configuration and insert your Liketly Username, API Token.
5. Done.
