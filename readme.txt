Liketly Bar

== Description ==

+ This plugin will show <a href="http://liket.ly/likebutton/">LiketLy Bar</a> on your website
+ Auto Post a <a href="http://www.liketly.com">LiketLy</a> Update  when you publish/update your Wordpress blog.
+ The Latest Way to let Visitors know Your Live News 
+ Demo: <a href="http://www.expertcustomize.com/demo/arowana/">Arowana Blog</a>

== Installation ==

1. Upload plugin zip file to the `/wp-content/plugins/` directory
2. Register to <a href="http://liket.ly">Liket.ly</a> to update your news.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Appearance > Liketly Bar Configuration and insert your Liketly Username, Token.
